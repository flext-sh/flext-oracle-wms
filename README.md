# FLEXT Oracle WMS

Biblioteca de integracao com Oracle WMS para acesso a dados operacionais de armazem.

Descricao oficial atual: "Enterprise Oracle WMS client library for FLEXT data integration platform".

## O que este projeto entrega

- Consome endpoints de operacao logistica do WMS.
- Padroniza coleta de dados de estoque e movimentacao.
- Apoia integracoes de supply chain com dados Oracle.

## Contexto operacional

- Entrada: chamadas para APIs Oracle WMS.
- Saida: dados operacionais para pipelines e servicos.
- Dependencias: credenciais WMS e conectividade com ambiente Oracle.

## Estado atual e risco de adocao

- Qualidade: **Alpha**
- Uso recomendado: **Nao produtivo**
- Nivel de estabilidade: em maturacao funcional e tecnica, sujeito a mudancas de contrato sem garantia de retrocompatibilidade.

## Diretriz para uso nesta fase

Aplicar este projeto somente em desenvolvimento, prova de conceito e homologacao controlada, com expectativa de ajustes frequentes ate maturidade de release.
